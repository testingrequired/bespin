# Development

## Yarn

This project uses yarn's workspaces so it will be required.

## Install

`yarn install`

## Build

Build all packages

`yarn build`

## Tests

Run tests in all packages

`yarn test`

## Verify

Runs the build and test

`yarn verify`

## CLI

To run the bespin configured in the example package. Great for quickly playing with the CLI.

```bash
$ yarn bespin
$ yarn bespin -h
```
